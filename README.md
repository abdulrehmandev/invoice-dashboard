# Invoice Dashboard
Invoice Dashboard built with Next.js Server Actions using Drizzle ORM

<img src="app/opengraph-image.png">

### Tech Stack
- Next.js
- Tailwind CSS
- Next Auth
- Drizzle ORM
- Vercel PostgreSQL

## Getting Started
First, setup the environment variables using `.env.example`

Second, run the development server:
```
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
